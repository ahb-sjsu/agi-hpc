# Beyond the Heliopause — Worldbuilding Index

This is the master TOC for the campaign's wiki bible. Entries
under each branch are markdown files with YAML frontmatter; the
build_bible.py script walks the tree and emits chunks for the
ARTEMIS / SIGMA-4 RAG.

Visibility convention (frontmatter ``artemis`` / ``sigma4``):
- **known** — the AI may discuss this on the record.
- **unknown** — the AI knows OF it but cannot recall details.
- **forbidden** — the AI will not refer to this in any reply.
  Keeper-only material; the validator's forbidden-phrase list
  also blocks the title from leaking.

## Locations

Each is a multi-entry sub-tree. Target ~200 pages per location
across 80–150 atomic entries, written incrementally.

- **luna/** — industrial belt, lower-g gateway, refugee culture
